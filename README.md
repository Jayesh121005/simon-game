# 🎮 Simon Game

An interactive memory-based game inspired by the classic **Simon Game**, where players must repeat an increasingly complex sequence of colors and sounds.

---

## 🌐 Live Demo

👉 https://jayesh121005.github.io/simon-game/

---

## 📌 Features

* 🎨 Interactive color buttons (Red, Blue, Green, Yellow)
* 🔊 Sound feedback for each button
* ❌ Game over sound & animation
* 📈 Level progression system
* 🧠 Memory-based gameplay
* 🏆 High score tracking

---

## 🧠 How to Play

1. Click anywhere to start the game
2. Watch the sequence shown by the system
3. Repeat the sequence by clicking buttons
4. Each level adds one new step
5. Game ends if you click the wrong color

---

## ⏱ Game Logic

* Uses **arrays** to store:

  * Game sequence (`gamePattern`)
  * User input (`userClickedPattern`)
* Compares user input step-by-step
* Generates random sequence using `Math.random()`
